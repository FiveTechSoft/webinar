# Webinar

[![GitHub last commit](https://img.shields.io/github/last-commit/FiveTechSoft/webinar)](https://github.com/FiveTechSoft/webinar/commits/master)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Repositorio de pruebas y ejemplos prácticos realizados durante sesiones de webinar. Este espacio contiene el material de apoyo, ejercicios y demostraciones utilizados en directo para explorar herramientas, flujos de trabajo y buenas prácticas de desarrollo.

## 📂 Estructura

```
webinar/
├── .claude/        # Configuración y memoria de Claude Code
├── texto.txt       # Archivo de ejemplo para pruebas
└── README.md       # Este documento
```

## 🎯 Propósito

- Servir como sandbox para demostraciones en tiempo real
- Ilustrar flujos de trabajo con Git, GitHub y Claude Code
- Probar integraciones y automatizaciones durante las sesiones

## 🚀 Requisitos

- [Git](https://git-scm.com/) 2.x+
- [GitHub CLI](https://cli.github.com/) (opcional)
- [Claude Code](https://claude.ai/code) (para las sesiones de IA)

## 📝 Notas

El contenido de este repositorio es efímero y de carácter educativo. Cada sesión puede añadir, modificar o eliminar archivos según el temario cubierto.
